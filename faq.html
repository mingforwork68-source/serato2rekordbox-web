常見問題

Serato → Rekordbox 轉換工具(離線網頁版)

這個工具跟 Serato、Pioneer DJ(rekordbox)官方有合作或授權關係嗎?

沒有。這是個人開發、免費提供的非官方工具,與 Serato Inc.、AlphaTheta Corporation(rekordbox / Pioneer DJ)並無任何隸屬、合作或授權關係,詳見免責聲明。

我的音樂檔案會被上傳到伺服器嗎?

不會。整個轉換過程 100% 在你自己的瀏覽器裡執行,音樂檔案與路徑資訊都不會透過網路傳送到任何伺服器,詳見隱私權政策。

支援哪些音樂檔案格式?

MP3、FLAC、WAV、AIFF、M4A/ALAC。

Hot Cue、Loop 資訊轉換的準確度如何?

工具會直接讀取 Serato 原始的 Cue 資料並對應到 Rekordbox 的格式,絕大多數情況能正確轉換。不過兩邊軟體的 Cue 系統設計不完全相同(例如顏色顯示、數量上限等細節),建議匯入 Rekordbox 後實際比對一次,確認結果符合預期。

為什麼還要手動輸入音樂資料夾路徑,不能自動偵測嗎?

基於瀏覽器安全限制,網頁沒辦法自動讀取資料夾在你電腦上的實際絕對路徑,只能取得檔案本身的內容與檔名。所以需要手動貼上路徑文字,產生的 XML 才能讓 Rekordbox 正確找到音樂檔案的位置。

轉換完成後,要怎麼把 XML 匯入 Rekordbox?

回到轉換工具首頁,下方步驟⑤有完整的圖文教學(含截圖),照著操作即可。

下載的報告裡,ERROR 或 NOCUE 是什麼意思?

ERROR 代表這首歌在轉換時發生錯誤(例如檔案格式不支援或已損毀);NOCUE 代表沒有讀取到這首歌的 Serato Cue 資訊(可能是這首歌原本在 Serato 裡就沒有設定過 Hot Cue/Loop)。可以下載 report.txt 查看詳細清單。

使用前為什麼建議先備份?

本工具設計上只會「讀取」你的檔案,不會覆寫或修改任何原始檔案。但任何操作前先備份都是好習慣,可以避免與本工具無關的意外(例如硬碟或作業系統問題)造成資料遺失。

這個工具需要付費嗎?

完全免費。原始碼採用 MIT 授權,公開在 GitHub 上。

支援手機或平板使用嗎?

建議使用桌面版瀏覽器(如 Chrome、Edge、Firefox)。因為需要選取整個音樂資料夾,手機瀏覽器普遍不支援這項功能。

Windows 和 Mac 都能用嗎?

都可以。這是純網頁工具,只要瀏覽器支援即可,不需要另外安裝任何軟體。

使用時遇到問題,或有功能建議,要怎麼回報?

歡迎到 GitHub Issues 頁面回報,連結也在每個頁面的頁尾。
