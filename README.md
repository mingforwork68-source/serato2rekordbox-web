# Serato → Rekordbox 轉換工具

## 繁體中文

**Serato → Rekordbox 轉換工具**是一個離線網頁版小工具,幫 DJ 把 Serato 的播放列表(crate)、Hot Cue、Loop 等資訊轉換成 Rekordbox 可以匯入的 XML 檔案。整個轉換過程完全在你自己的瀏覽器裡執行,音樂檔案與路徑資訊都不會透過網路傳送或上傳到任何伺服器,關掉分頁資料就清除。支援繁體中文、简体中文、English 三種介面語言。

**功能:**
- 讀取 Serato DJ 的 `.crate` 播放列表,還原歌曲順序與播放列表資料夾結構
- 轉換 Hot Cue、Loop、Memory Cue 等 Cue point 資訊到 Rekordbox
- 支援 MP3、FLAC、WAV、AIFF、M4A/ALAC 音樂檔案格式
- 產生可直接匯入 Pioneer DJ rekordbox 的標準 XML 檔案
- 100% 瀏覽器端執行,無需安裝、無需上傳音樂檔案,保護隱私
- 免費、開源(MIT License)

## English

**Serato → Rekordbox Converter** is an offline, browser-only tool that helps DJs convert Serato playlists (crates), Hot Cues, and Loops into an XML file that Rekordbox can import. The entire conversion runs inside your own browser — music files and path information are never sent over the network or uploaded to any server, and everything is cleared once you close the tab. Available in Traditional Chinese, Simplified Chinese, and English.

**Features:**
- Reads Serato DJ `.crate` playlists, restoring track order and playlist/folder structure
- Converts Hot Cues, Loops, and Memory Cues into Rekordbox-compatible cue points
- Supports MP3, FLAC, WAV, AIFF, and M4A/ALAC audio files
- Generates a standard XML file ready to import into Pioneer DJ rekordbox
- Runs 100% in your browser — no install, no music file uploads, privacy-friendly
- Free and open source (MIT License)
