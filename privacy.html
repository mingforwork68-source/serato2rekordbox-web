Serato → Rekordbox 轉換工具(離線網頁版)

這個工具怎麼處理你的資料

本工具是一個純前端(client-side)網頁應用程式。你選取或拖入的音樂檔案、Serato 播放列表(.crate)資料,全部只會在你自己電腦上的瀏覽器裡被讀取與解析,用來產生 Rekordbox XML 檔案。整個過程中,這些檔案內容不會透過網路上傳、傳送或儲存到任何伺服器,包含本網站本身的伺服器在內。

你關閉分頁或重新整理頁面後,所有已讀取的資料就會從瀏覽器記憶體中清除。

本機儲存(localStorage)

為了方便使用,本工具會用瀏覽器的 localStorage 記住你輸入過的「音樂資料夾絕對路徑」文字,純粹是為了下次不用重新輸入。這筆資料只存在你自己的瀏覽器裡,不會被傳送出去,你可以隨時透過瀏覽器設定清除網站資料來移除它。

Cookie 與廣告

本網站使用 Google AdSense 刊登廣告。Google 及其廣告合作夥伴可能會使用 Cookie 或類似技術,依你先前造訪本網站及其他網站的紀錄,投放個人化廣告。這部分的資料蒐集與使用由 Google 主導,與本工具的「轉檔功能」完全分開(轉檔功能仍是 100% 在你自己的瀏覽器裡執行,不會上傳任何音樂檔案或路徑資訊)。你可以前往 Google 廣告設定 管理或關閉個人化廣告,也可以參考 Google 合作夥伴網站如何使用 Cookie 的說明。

網站託管

本網站由 GitHub Pages 提供靜態網頁託管。GitHub 可能依其自身服務需求,記錄基本的伺服器存取紀錄(例如 IP 位址、瀏覽器資訊),這部分不屬於本工具的控制範圍,詳情請參考 GitHub 的隱私權聲明。

未來若有變更

若本工具未來新增任何會蒐集資料、使用 Cookie 或投放廣告的功能,會在此頁面更新說明。
